# Memory Game Application

## 📌 Description
This project implements a Memory Card Game using HTML, CSS, and JavaScript. The application allows users to flip cards and match pairs, demonstrating DOM manipulation, event handling, and state management.

## 🚀 Features
- Card flipping functionality using JavaScript event handling
- Shuffle algorithm to randomize card positions at game start
- Game state tracking (flipped cards, matched pairs)
- Matching logic with delayed reset for unmatched cards
- Restart functionality to reset and reshuffle the board
- Responsive design for both desktop and mobile devices

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (DOM Manipulation)

## ▶️ How to Run
1. Open `index.html` in a browser
2. Click cards to flip and find matching pairs
3. Use the restart button to reset the game
